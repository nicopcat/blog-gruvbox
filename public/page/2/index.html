2022-04-23

Codewar 上有一题非常简单的题：

写一个函数，给任意一个正整数 n，求和。

Example:

2 -> 1 + 2  // output: 3
5 -> 1 + 2 + 3 + 4 + 5 //  output: 15

【小学生解法】

先用小学就学过等差数列的累加计算公式“( 首项 + 末项 ) * 项数 / 2”计算一下:

2 -> (1+2)* 2/2 = 3  ✔
5 -> (1+5)* 5/2 = 15 ✔

本小学生做对了！
